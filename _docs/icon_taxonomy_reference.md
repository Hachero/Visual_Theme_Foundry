# Icon Taxonomy Reference

## Purpose

This document is the structural reference for grouping generic icon names.
Use it as the source taxonomy. Use icon_reference_groups.md for actual filename mappings.

## 1) Core Web-App Icon Groups

### 1.1 Main-Primary (18)

The main-primary set defines the highest-frequency icon intents used across most web applications:

1. Home
2. Search
3. Menu
4. Close
5. Add
6. Edit
7. Delete/Remove
8. Save
9. Settings
10. User/Profile
11. People/Team
12. Notifications/Alerts
13. Message/Chat
14. Calendar/Schedule
15. File/Document
16. Folder
17. Upload/Download
18. Share

## 3) Matching Convention

When mapping files to generic names:

- Match by lowercase filename string containment.
- Keep generic names semantic and vendor-neutral.
- Allow a file to belong to multiple groups when a name overlaps contexts.
- Treat this taxonomy as practical IA for icon discovery, not strict ontology.
